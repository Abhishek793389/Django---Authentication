About User Authentication System

The User Authentication System in a web application is essential for ensuring that only authorized users can access certain features or pages. It involves verifying a user's identity using credentials such as a username and password. Below is an overview of how our authentication and session management system works.

Key Features of the Authentication System

  • Registration: Users can sign up by providing their personal details like username, password, and email.
  • Login: Registered users log in by entering their valid credentials.
  • Session Management: A session is created after a successful login, allowing users to stay authenticated across multiple pages.
  • Dashboard Access: After logging in, users are redirected to their personal dashboard where they can manage their activities.

Session Management and User Activities

Our system uses Djangoâ€™s built-in session management to keep track of logged-in users. Once authenticated, the user can perform the following activities:

  • Add Blog: Users can create new blog posts with a title and content.
  • View Blog: Users can view their own blogs and those created by others.
  • Update Blog: Users can edit the content of their existing blogs.
  • Delete Blog: Users can remove their blogs from the platform.
  • Change Password: Users can update their password from the dashboard for better security.
  • Logout: When users log out, the session ends, and they must log in again to access their dashboard.
  • Delete User Account: Users can delete their accounts, which removes all associated data.

Security Features

  • Password Hashing: User passwords are securely stored using hashing algorithms.
  • Session Expiration: Sessions expire after a period of inactivity to prevent unauthorized access.
  • CSRF Protection: Cross-Site Request Forgery protection is enabled for secure form submissions.

Technologies Used

  • Django Framework: Used for building the backend authentication system.
  • HTML & CSS: For the frontend interface and styling.
  • MySQL Database: For storing user data and blog information.

This user authentication system ensures that users can securely manage their data and perform actions such as adding, updating, and deleting blogs, while keeping their personal information safe.
